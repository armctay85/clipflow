# ClipFlow AI

Video to Social Content — Automated

## Quick Start

```bash
cd src
npm install
npm start
```

## API Endpoints

- `GET /health` - Health check
- `POST /api/extract` - Extract YouTube transcript
- `POST /api/generate` - Generate social content
- `POST /api/transform` - Full pipeline
- `POST /api/waitlist` - Join waitlist

## Environment Variables

Copy `.env.example` to `.env` and fill in your keys.
